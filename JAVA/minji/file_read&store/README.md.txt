     file read, store 는 파일에 써서 저장하고 그 파일 내용을 읽어오는 프로그램입니다.

     실행방법 

        $ javac file_read.java
        $ javac file_store.java

        $ java file_read


	실행방법을 자세히는 모르겠지만 main 은 file_read 에 있습니다.
